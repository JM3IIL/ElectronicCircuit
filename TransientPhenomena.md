# 過渡現象

## RL回路
過渡状態における、電流 : I(t) [A]

$I(t) = \frac{V}{R} (1-e^{-\frac{Rt}{L}})$

時定数 : $τ = \frac{L}{R} [s]$

単位変換 : $\frac{L[H]}{R[Ω]} = \frac{[V][s]}{[A]} / \frac{[V]}{[A]}$

https://detail-infomation.com/rl-circuit-time-constant/


## RC回路
過渡状態における、Cの電圧 : V_C(t) [V]

$V_C(t) = V (1-e^{-\frac{t}{RC}})$

時定数 : $τ = CR [s]$

単位変換 : $C[F] × R[Ω] = \frac{[A][s]}{[V]} × \frac{[V]}{[A]}$
