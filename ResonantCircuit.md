# 共振回路

## LC回路
- コイル : L [H]
- コンデンサ : C [F]

直列共振条件
$2πfL = \frac{1}{2πfC}$
並列共振条件
$\frac{1}{2πfL} = 2πfC$

共振周波数
$f = \frac{1}{2\pi\sqrt{LC}} [Hz]$
