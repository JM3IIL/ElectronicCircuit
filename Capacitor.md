# コンデンサ

コンデンサの「電圧・電流と時間」の関係
https://cc.cqpub.co.jp/system/contents/2253/

静電容量(キャパシタンス)
コンデンサが蓄えることができる電荷の量

Q(クーロン)
電荷量の単位:電流1[A]を1[s]で蓄える電荷量1[Q]
[Q]=[A・s]

F(ファラド)
静電容量の単位:1[F]のコンデンサ、1[Q]の電荷量で、電極間の電位差は1[V]


- 導体板間の誘電体の誘電率 $ε [F/m]$
- 導体板の面積 $S [m^2]$
- 導体板間の距離 $d [m]$を除して求めます。
- 真空の誘電率 $ε_o [F/m]$
- 導体板間の誘電体の比誘電率 $ε_r$

$C = εS/d [F]$
$C = ε_o ε_r S/d [F]$
